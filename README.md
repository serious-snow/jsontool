# jsontool

## Tool list

* json --> go struct (comment supported)
* json --> protobuf (comment supported)
* swagger --> markdown

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
